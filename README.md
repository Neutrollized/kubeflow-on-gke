# Kubeflow

Made up components which you can install the entire ecosystem or just pick the ones you use. Kubeflow components are:

- KServe
- Katib (i.e. AutoML)
- Model Registry
- Notebooks
- Pipelines
- Spark Operator
- Trainer
- Dashboard

 
 The tasks listed in `Taskfile.yaml` are meant to be for standalone installs.


## Full install
For full install, you can consider using the [Kubeflow manifests repository](https://github.com/kubeflow/manifests).  I have included a `kustomization.yaml` file
